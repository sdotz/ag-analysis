VDOT vs Age Grade Analysis

Masters All-Age Best performances — discrepancy between Jack Daniels VDOT and Age-Grade %

How calculations work
  • VDOT — Jack Daniels aerobic-capacity score. v = dist ÷ time (m/min), VO₂ = −4.60 + 0.182258·v + 0.000104·v², pct_max = fraction of VO₂max sustainable for that race duration, VDOT = VO₂ ÷ pct_max. Higher = stronger aerobic engine. Age-blind — no correction for the runner's age.
  • AG% — WMA Age Grade percentage. AG% = (age-group standard time ÷ actual time) × 100. The standard is the open world record divided by an age factor from WMA lookup tables. 100% = world record for that age and gender. Explicitly age-adjusted.
  • Δ rank — Within each gender + distance group, both metrics are converted to 0–100 percentile ranks. Δ rank = AG% rank − VDOT rank. Positive means AG% places the runner higher; negative means VDOT does.
  • Spearman ρ — Rank correlation between the AG% and VDOT orderings within a group. 1.0 = identical ranking, 0 = no agreement, −1 = reversed.
